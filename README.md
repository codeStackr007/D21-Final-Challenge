# Conquering Responsive Layouts - Final Challenge

Final project for Kevin Powell's "Conquering Responsive Layouts" course.

## Built With

- HTML5
- CSS3 (Flexbox, Custom Properties, Media Queries)
- JavaScript (Mobile menu toggle)
- Google Fonts (Oswald + Montserrat)
- Font Awesome 7

## Features

- Mobile-first workflow
- Hamburger menu (mobile only)
- Fully responsive (mobile, tablet, desktop)
- BEM naming convention
- Semantic HTML
- Accessible focus states

## Breakpoints

| Device  | Width         |
| ------- | ------------- |
| Mobile  | 0 - 767px     |
| Tablet  | 768px - 999px |
| Desktop | 1000px+       |

## Course

[Conquering Responsive Layouts](https://courses.kevinpowell.co/view/courses/conquering-responsive-layouts) by Kevin Powell

## Setup

No build tools needed. Open `index.html` in a browser.
